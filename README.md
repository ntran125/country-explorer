# Country Explorer

## Project Description
Country Explorer is a simple website where users can search for countries and learn basic information about them. The app uses the REST Countries API to display country data such as the flag, capital, population, and region.

This project was built using:
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Target Browsers
This project works on:
- Google Chrome
- Microsoft Edge
- Firefox
- Safari
- iPhone browsers
- Android browsers
